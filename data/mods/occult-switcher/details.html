What does this mod do?

This mod lets you change any sim's occult type during live mode via a cheat interaction. You can switch between: Human (Non-Occult), Alien, Vampire, Mermaid, Spellcaster, and Werewolf. Hybrid occult types are not supported.

PlantSims, Ghosts, Skeletons, and Servos are not yet supported.

How do I use it?

To access the cheat interaction, you must first enable testing cheats with the testingcheats on cheat code (note that this cheat is per-save). When testing cheats are on, shift-click the sim you'd like to edit, and select the Change Occult Type... interaction.

To enter a cheat code: (1) Press CTRL + SHIFT + C, (2) Type the cheat code, (3) Press enter.


What happens when I use this cheat?

The chosen sim will immediately turn into the selected occult, leaving their previous occult form(s) behind. However, occult appearances and progression trees are not disturbed - you can switch back to the original occult form at any time and pick up where you left off.

Important: If used on a hybrid occult, you will not be able to get their hybrid status back.

What if I don't have all occult packs?

No problem. If you don't have the packs for certain occult types, they simply won't appear as options.


Occult Switcher is intended for those who want a lightweight mod that adds cheats for vanilla occult behavior. If you want a mod that alters occult behavior and supports hybrids, consider Occult Hybrid Unlocker & Stabilizer by Iced Cream and TwelfthDoctor1. It is not recommended to use both mods at the same time.
